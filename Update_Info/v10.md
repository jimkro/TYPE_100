# 🛠️ 開發迭代紀錄：v10.0

## ⚖️ 平衡調整與動態難度 (Balance & Dynamic Scaling)
`狀態：數值校優與易用性提升`

v10 是目前最均衡的版本。我們針對前幾版中過於強大的武器進行了限制，並引入了隨玩家等級成長的敵人，確保遊戲中後期依然保有壓迫感。

### 核心功能進化
* **動態難度曲線 (Dynamic Difficulty)**：
    * **等級加成機制**：怪物的移動速度不再僅由關卡決定，現在會隨著玩家等級進行動態加成（`baseSpeed + level * 0.05`）。
    * **後期挑戰**：這意味著玩家雖然變強了，但怪物的威脅性也會同步上升。
* **武器平衡性修正 (Weapon Tuning)**：
    * **燃燒瓶 (Molotov) 削弱**：修正了 v6-v9 中燃燒瓶過於強大的問題。縮短了火海持續時間（500ms -> 250ms），並降低了削減怪物字數的判定頻率，增加戰術使用的門檻。
* **操作與 UX 優化 (Quality of Life)**：
    * **切換提示**：現在使用 `Tab` 切換武器時，角色頭頂會彈出動態浮動文字提示當前武器名稱（如 `Pistol`, `Bow`）。
    * **升級快捷鍵**：在升級選單中，新增鍵盤快捷鍵 `1` / `2` / `3` 快速選擇強化卡片，大幅提升了操作連貫性。
* **視覺與音效回饋強化**：
    * 優化了攻擊效果的渲染層級，確保不同武器的彈道視覺不會互相遮擋。
    * 微調了受傷時的視覺抖動（Camera Shake）感。

### 程式碼底層優化
* **碰撞偵測精細化**：重構了投射物與怪物的碰撞矩陣，提升了大規模戰鬥時的運算效率。
* **狀態重置邏輯**：修正了 `resetGame()` 時，部分武器冷卻時間未歸零的邊際案例。

> **開發筆記**：v10 是這場 Vibe Coding 實驗的一個重要里程碑。我們從純粹的功能堆疊轉向了「遊戲體驗」的深度打磨。透過數值拉鋸與 UX 微調，遊戲的沉浸感獲得了質的提升。
