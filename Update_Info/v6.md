# 🛠️ 開發迭代紀錄：v6.0

## ⚔️ RPG 系統與武器多樣化 (RPG Systems & Weapon Variety)
`狀態：核心玩法演進`

v6 引入了經驗值與升級機制，將單一的戰鬥行為演變為具備「成長回饋」的 RPG 模式，並大幅擴充了武器庫，讓戰鬥具備組合策略。

### 核心功能進化
* **RPG 升級系統 (Leveling System)**：
    * **經驗值機制**：擊殺怪物現在會掉落/獲得 XP，並設有動態的升級經驗門檻（xpNext）。
    * **升級選單 (Upgrade Modal)**：升級時遊戲會自動暫停，並跳出三張隨機生成的「強化卡片」。
    * **屬性強化**：玩家可選擇提升 `Max HP`、`移動速度` 或進行 `生命值回補`。
* **多樣化武器庫實作 (Expanded Arsenal)**：
    * 系統不再只有單一攻擊方式，改為由多種投射物（Projectiles）驅動的武器系統：
        1. **Pistol (手槍)**：基礎武器，單體快速打擊。
        2. **Bow (弓箭)**：具備直線穿透特性，可一次傷害路徑上的多個敵人。
        3. **Grenade (手榴彈)**：範圍爆炸，具備「削弱字數」的特殊效果。
        4. **Molotov (燃燒瓶)**：產生持續性的燃燒地帶（Fire Zones），提供區域控場。
* **投射物物理引擎 (Projectile Physics)**：
    * 為不同武器實作了各自的飛行速度、存活時間（Life）與碰撞判定邏輯。
* **介面 UI 2.0 (Enhanced Overlay)**：
    * **XP 進度條**：在頂部實作了視覺化的經驗值條。
    * **升級介面**：採用全螢幕遮罩設計，並實作卡片懸浮效果與按鈕點擊互動。
* **戰場狀態管理 (State Management)**：
    * 實作了「暫停/運行」切換邏輯，確保玩家在選擇升級卡片時不會受到怪物攻擊。

### 視覺與特效調整
* **武器專屬特效**：針對不同武器設計了專屬色彩（綠色箭頭、黃色爆炸、橘色火海）。
* **UI 分層**：優化了 DOM 層級，將 UI Overlay 與 Canvas 渲染完全分離，提升介面響應速度。

> **開發筆記**：v6 是遊戲性的分水嶺。透過「隨機強化」與「多武器選擇」，遊戲從原本單調的避難，變成了尋求最強 Build 的策略挑戰。這是 Vibe Coding 過程中，遊戲複雜度提升最明顯的一版。
