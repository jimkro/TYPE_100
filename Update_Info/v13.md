# 🛠️ 開發迭代紀錄：v13.0

## 📚 常用詞庫與難度重構 (English 1k Dictionary)
`狀態：內容豐富化與難度平衡`

v13 是一次內容層面的重大升級。我們引入了精心篩選的「1000 個常用英文單字庫」，取代了原本純隨機的字母組合，讓遊戲過程同時兼具詞彙練習的效果。

### 核心功能進化
* **常用單字庫實作 (English 1k Dictionary)**：
    * **邏輯大改**：廢棄了單純的隨機字元生成，改為從內建的 1000 個高頻英文單字中進行抽取。
    * **學習與記憶**：玩家在擊殺怪物的同時，能練習真正的單字拼寫（如：`people`, `question`, `river` 等），大幅提升了遊戲的教育價值。
* **難度與速度動態平衡**：
    * **等級加成優化**：怪物的移動速度加成係數從 v10 的 `0.05` 提升至 `0.12`。
    * **生存挑戰**：隨著玩家等級提升，怪物靠近的速度會顯著加快，要求玩家必須在單字長度增加的同時維持拼寫速度。
* **物件生命週期優化**：
    * 微調了投射物與火海區域（Fire Zones）的生命週期邏輯，確保在大量單字同時出現時，渲染效能依然維持在 60 FPS。
* **UI 與互動一致性**：
    * 延續了 v12 的軟碰撞（Soft Separation）物理系統，確保在常用單字模式下，怪物依然會保持間距，不會因單字過長而互相遮擋。
    * 武器狀態與冷卻提示現在會更精確地對應單字庫的難度進階。

### 內容更動
* **字串生成邏輯**：系統會自動根據當前關卡（Stage）過濾詞庫中長度合適的單字，確保難度曲線的平滑。
* **視覺修正**：微調了角色頭頂輸入框的垂直座標，以適應長度較長的英文單字顯示。

> **開發筆記**：v13 讓這款遊戲從「打字練習器」轉型為一個具備「語言學習」潛力的作品。透過引入真實單字庫，每一次的擊殺都變成了一次有效的詞彙記憶，這是我在 Vibe Coding 過程中，對遊戲「價值」最深的一次思考與實踐。
