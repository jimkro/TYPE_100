# 🛠️ 開發迭代紀錄：v1.0

## 🚀 核心架構建立 (Initial Architecture)
`狀態：技術原型驗證`

此版本的主要目標是驗證「全文字輸入控制」的可行性，並建立底層的遊戲循環系統。

### 核心功能實作
* **指令緩衝系統 (Command Buffer)**：實作了基礎的鍵盤監聽邏輯，將玩家輸入的字元緩存並與指令集進行即時比對。
* **位移邏輯轉換**：定義了 `up`, `down`, `left`, `right` 四組核心指令，成功將打字輸入轉換為 Canvas 座標的位移動作。
* **動作功能實作**：新增 `attack` 指令，並實作了第一個以玩家為中心的圓形範圍攻擊特效（由 `player.isAttacking` 狀態驅動）。
* **即時視覺反饋**：
    * **浮動文字系統 (Floating Text)**：用於顯示指令觸發成功的結果。
    * **玩家輸入顯示**：在角色頭頂實作即時輸入顯示，確保操作的直覺性與準確性。
* **環境渲染基礎**：使用 HTML5 Canvas 建立 800x600 的網格戰場，確立後續開發的視覺基準線。

> **開發筆記**：v1 成功驗證了打字與即時動作結合的操作手感。雖然目前僅有單一玩家方塊，但基礎的「輸入-判斷-反應」鏈條已經穩定運作。
