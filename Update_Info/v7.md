# 🛠️ 開發迭代紀錄：v7.0

## 🎨 UI 重構與視覺演進 (Combat Evolved UI)
`狀態：介面與體驗優化`

v7 標誌著遊戲從「開發者工具風格」向「正式遊戲介面」的轉型。我們廢棄了簡單的 Canvas 文字疊加，改用完整的 HTML/CSS 疊層來建構動態 UI 系統。

### 核心功能進化
* **全新 UI 疊層系統 (HTML/CSS UI Layer)**：
    * **結構化佈局**：利用 Flexbox 重新規劃了頂部資訊欄，將 `HP`、`XP`、`得分` 與 `關卡資訊` 進行模組化排列。
    * **視覺化進度條**：實作了具備 CSS Transition 平滑過渡效果的血條（HP Bar）與經驗條（XP Bar），提供更直覺的生存反饋。
* **關卡系統與資訊 (Stage Info)**：
    * 引入了「Stage」概念，並在 UI 中即時顯示當前關卡（如 Stage 1），為後續的難度遞進建立框架。
* **武器資訊顯示**：
    * 在 UI 底部（或側邊）新增了目前所選武器的文字提示，解決了 v6 中玩家容易忘記當前武器類型的問題。
* **小地圖 (Minimap) 精緻化**：
    * 重新調整小地圖至右上角，並優化了背景遮罩與邊框，使其在混亂的背景中依然具備高度可讀性。
* **操作與互動優化**：
    * **禁止選取 (user-select: none)**：實作全域禁用選取功能，防止玩家在激烈的打字過程中誤觸滑鼠導致文字被選取，大幅提升了操作的流暢度。
    * **字體渲染最佳化**：統一採用 `Courier New` 搭配 `text-shadow`，強化了文字在不同背景下的對比度。
* **底層效能調整**：
    * 優化了 `draw()` 函式與 DOM UI 的互動頻率，減少不必要的介面重新渲染，確保 60 FPS 的穩定輸出。

### 視覺調整
* **色彩方案更新**：優化了 HP（紅色）、XP（藍綠色）的飽和度，符合遊戲視覺心理學。
* **浮動特效優化**：微調了擊殺特效的漂浮速度與淡出時間，減少視覺干擾。

> **開發筆記**：v7 的重點在於「資訊的可視化」。透過專業的 UI 系統，玩家現在能非常精確地掌握自己的生存狀態與成長進度，這讓整體的遊戲感（Game Feel）提升了一個檔次。
