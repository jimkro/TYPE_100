# 🛠️ 開發迭代紀錄：v2.0

## 👾 戰鬥系統與生存機制 (Combat & Survival)
`狀態：遊戲性功能實作`

在 v1 的基礎上，此版本導入了敵對單位與勝負目標，正式將「生存挑戰」元素融入打字機制中。

### 核心功能進化
* **動態敵人系統**：實作了邊界生成演算法（四維隨機生成）與基礎追蹤 AI，使怪物會自動計算向量向玩家座標靠攏。
* **目標擊殺機制大改版**：
    * **邏輯轉變**：移除 v1 泛用的 `attack` 指令，改為「精確目標輸入」。
    * **字串綁定**：將隨機單字庫（WORD_LIST）綁定至敵對單位，玩家必須完全匹配特定單字方可將其消除。
* **計分系統 (Score System)**：根據單字長度實作加權計分邏輯（單字長度 × 10），並即時同步至頂部 UI 顯示。
* **物理邊界限制**：加入碰撞檢查邏輯，限制玩家座標必須在 Canvas 畫布範圍內，防止角色移出視窗。
* **介面優化 (UI/UX)**：
    * **易讀性強化**：在玩家輸入緩衝區後方增加半透明底框，提升在移動過程中的文字辨識度。
    * **資訊層級**：新增頂部狀態欄位，顯示當前累積分數與基本移動指引。

> **開發筆記**：v2 的核心演進在於「多工處理」。玩家現在必須在「位移逃生」與「精確擊殺」之間進行快速決策，這大幅提升了遊戲的挑戰深度與可玩性。
