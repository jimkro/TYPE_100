**Typing Survival** 是一款將「打字練習」與「類倖存者 (Survivor-like)」機制結合的極簡動作遊戲。這不僅測試你的打字速度，更考驗你在高壓環境下的策略選擇與反應。

本專案目前的開發目標是透過 **Vibe Coding** 進行快速迭代，目前的版本為 **v10 平衡性更新**。

## 🎮 遊戲核心機制

* **打字即行動**：輸入指令（如方向或單字）後按 `Space` 或 `Enter` 觸發行動。
* **動態難度**：怪物的基礎速度會隨等級提升（Level Bonus），讓你越後期越手忙腳亂。
* **多樣武器系統**：
    * 🔫 **手槍 (Gun)**：單體高傷，針對性擊殺。
    * 🏹 **穿透弓 (Bow)**：直線穿透所有敵人，清理大隊伍首選。
    * 💣 **虛弱雷 (Grenade)**：範圍縮減敵人的單字長度，讓他們更好對付。
    * 🔥 **燃燒瓶 (Molotov)**：在地面留下持續性的削字火海。
    * 💊 **急救包 (Heal)**：升級時可選的關鍵回復手段。

## 🚀 v10 更新亮點 (Balance Update)

這個版本專注於遊戲節奏的平衡與操作感的優化：

1.  **操作優化**：現在 `Space` 與 `Enter` 均可作為輸出確認鍵。
2.  **武器平衡**：
    * **燃燒瓶 (Molotov) 削弱**：火海持續時間大幅縮短 (500 -> 250)，且削字判定頻率變慢，增加遊戲挑戰性。
3.  **怪物強化**：
    * 提升了怪物的基礎速度，並新增了等級加成機制，隨著玩家等級越高，怪物移動越快。
4.  **視覺回饋**：
    * 切換武器時（使用 `Tab`），角色頭頂會顯示當前武器名稱。
    * 升級選單現在支援鍵盤快捷鍵 `1` / `2` / `3` 快速選擇卡片。

## 🕹️ 操作說明

| 動作 | 指令 |
| :--- | :--- |
| **向上移動** | 輸入 `up` |
| **向下移動** | 輸入 `down` |
| **向左移動** | 輸入 `left` |
| **向右移動** | 輸入 `right` |
| **攻擊怪物** | 輸入怪物頭頂顯示的單字 |
| **確認輸入** | 按下 `Space` 或 `Enter` |
| **切換武器** | 按下 `Tab` |
| **刪除錯字** | 按下 `Backspace` |

## 🏗️ 開發哲學：Vibe Coding

這個專案是透過與 AI 協作，以「直覺」與「手感」驅動的迭代產物。
* **高頻率迭代**：從 v1 到 v10，每一版都針對前一版的「不順手」進行微調。

## 📅 關卡設定

* **Stage 1**：基礎鍵位 (asdf...)，單字長度 1-3。
* **Stage 2**：上排擴充 (qwerty...)，單字長度 2-4。
* **Stage 3**：全鍵盤挑戰 (zxcv...)，單字長度 3-5。
* **Endless**：全單字庫挑戰，怪物速度隨時間無限提升。
