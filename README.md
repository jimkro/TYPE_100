![Tech Stack](https://img.shields.io/badge/tech-Vanilla_JS_%7C_Canvas-yellow?style=for-the-badge)

**Typing Survival** 是一款將「打字練習」與「類倖存者 (Survivor-like)」機制結合的生存動作遊戲。不同於傳統的 WASD 移動，這款遊戲要求玩家透過打字來控制位移、施放技能與擊殺敵人。

本專案完全採用 **Vibe Coding** 模式開發：不預設複雜架構，純粹靠靈感與 AI 協作進行高頻率迭代。

---

## 🕹️ 遊戲核心機制

* **全鍵盤操作**：
    * **位移系統**：輸入 `up`, `down`, `left`, `right` 並按下 `Space/Enter` 進行八方位移動。
    * **精確擊殺**：輸入怪物頭頂顯示的隨機單字以觸發攻擊。
    * **武器切換**：按下 `Tab` 鍵在不同武裝間即時切換。
* **RPG 成長系統**：擊殺怪物獲得經驗值，升級時可從三張隨機卡片中選擇強化（提升 HP、速度或解鎖武器）。
* **多樣化武器庫**：
    * 🔫 **Pistol (手槍)**：單體精準打擊。
    * 🏹 **Bow (穿透弓)**：直線貫穿，清理成排敵人。
    * 💣 **Grenade (虛弱雷)**：範圍縮減敵人的單字長度。
    * 🔥 **Molotov (燃燒瓶)**：產生持續性削字火海（v10 進行過平衡性調整）。
* **動態物理引擎 (v12)**：實作了怪物間的「軟碰撞排斥力」，防止敵人群體重疊，提供更真實的包圍感與戰術空間。

## 🛠️ 開發哲學：Vibe Coding

這個專案是「直覺驅動開發」的實驗場：
1. **快速迭代**：每個版本僅專注於解決當下「感覺不對」的地方。
2. **AI 協作**：利用語言模型作為思考夥伴，將抽象的手感需求轉化為具體的物理公式與程式碼。
3. **極簡部署**：堅持 Vanilla JS 單檔案架構，實現「隨處開啟、無需環境」的極致體驗。

## 🎮 如何開始

1. 下載或 Clone 本倉庫。
2. 直接在瀏覽器中開啟 `v12.html` 即可遊玩。
3. **操作 Tips**：
    * 升級時可按鍵盤數字 `1`, `2`, `3` 快速選卡。
    * 隨時注意小地圖上的紅點，避免被包圍。
    * 靈活切換武器（Tab）以應對不同密度的敵人。
